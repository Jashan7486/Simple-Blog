# Simple Blog

Minimal Node/Express blog where users can write posts (title + content) and view them on the homepage.

Getting started

1. Install dependencies

```bash
npm install
```

2. Run the app

```bash
npm start
```

3. Open http://localhost:3000 and create posts using "New Post".

Files added

- `app.js`: Express server and routes
- `views/`: EJS templates (`index`, `new`, `show`, partials)
- `public/styles.css`: basic styling
- `data/posts.json`: persisted posts (JSON)
